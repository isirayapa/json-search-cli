# json-search-cli

## Prerequisites

**Python 3**``

**Virtual Environment**

`sudo apt install virtualenv`

`virtualenv venv --python=python3`

**Activate the python environment**

`source ./venv/bin/activate`

**Install Dependencies**

`pip install -r requirements.txt`

**Run application**

`python index.py`

**Run tests**

`pytest`
